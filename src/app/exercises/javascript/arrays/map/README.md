# Arrays

## Introducción

JavaScript tiene métodos de arrays como `map`, `filter`, `reduce`, `find` que son usados muy comúnmente.

## Requisitos

Dada una lista de números, debería devolver una nueva lista de números elevados al cuadrado. [1, 2, 3] ➡️ [1, 4, 9]
